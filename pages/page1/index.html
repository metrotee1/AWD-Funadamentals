The 4 Fundamentals of Coding
Variables
Variables are used to store data that can be used and modified later in the program.
Control Structures
Control structures determine the flow of execution in a program, such as loops and conditionals.
Data Structures
Data structures are used to organize and store data, such as arrays, lists, and trees.
Algorithms
Algorithms are step-by-step procedures or formulas used for solving a problem or performing a task.
